##  Installation

### Using Yarn (Recommended)

Clone repository

git clone https://github.com/dhirajkumargup/Portfolio.git

Go to folder

cd Portfolio

Install dependencies

yarn install

Run project

yarn dev

Build project

yarn build

Preview build

yarn preview

---

### Using npm (Alternative)

Install dependencies

npm install

Run project

npm run dev

Build project

npm run build

Preview build

npm run preview
